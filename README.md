# Shopping System for Minimarket

## Overview

The **Shopping System for Minimarket** is a sample project designed to demonstrate **Domain-Driven Design (DDD)** principles. It models a small-scale retail system where customers can browse products, place orders, and manage inventory.

This project focuses on **clean separation of concerns**, using DDD layers: Domain, Application, Infrastructure, and Interface.

---

## Features

* **Product Management** – add, update, and remove products.
* **Customer Management** – register and manage customer information.
* **Order Processing** – place orders, calculate totals, and track status.
* **Inventory Management** – adjust stock automatically when orders are placed.
* **Domain Events** – events triggered by important actions like `OrderPlaced` or `StockUpdated`.
